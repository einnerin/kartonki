## Цель

Привести **все** наборы к единому стандарту:

| Критерий | Требование |
|----------|-----------|
| Размер | Ровно **25 слов** |
| Тема | Все слова однозначно относятся к **одной теме** |
| Уровень | Все слова **близкой сложности** (например, A1–A2 или B1–B2) |
| Синонимы | Запрещены — два слова с одним смыслом в одном наборе бессмысленны |
| Однокоренные | Запрещены — `run`/`runner`, `day`/`daily` не несут обучающей ценности |

---

## Трёхуровневая структура (для богатых тем)

Если тема охватывает слова от A1 до C2 — она **разбивается на три набора**:

```
Тема: основы      → COMMON/UNCOMMON  (A1–A2)   — 25 слов
Тема: продвинутый → RARE             (B1–B2)   — 25 слов
Тема: углублённо  → EPIC/LEGENDARY   (C1–C2)   — 25 слов
```

Каждый из трёх — самостоятельный набор, ровно 25 слов.

---

## Правила рефакторинга (ОБЯЗАТЕЛЬНО)

1. **Читай набор целиком** прежде чем что-то менять.
2. **Убирай слова не своего уровня** — если в "основах" есть EPIC-слово, оно туда не подходит.
3. **Заменяй, а не переименовывай** — нельзя менять `rarity = "EPIC"` → `"RARE"` на том же слове. Сложность слова объективна, метка её не меняет. Нужно заменить слово на другое, реально подходящее по уровню.
4. **Убранные слова не теряй** — они уходят в набор своего уровня (продвинутый/углублённо), существующий или новый.
5. **Набор после правки = ровно 25 слов.** Проверяй: `grep -c "setId = N," файл.kt`
