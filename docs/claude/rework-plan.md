# Рефакторинг наборов слов — план и прогресс

> **ЧИТАЙ ЭТОТ ФАЙЛ ПЕРВЫМ** перед любой работой с наборами слов.

## Стандарт набора

| Критерий | Требование |
|----------|-----------|
| Размер | Ровно **25 слов** |
| Тема | Все слова — одна тема |
| Уровень | Не шире 2 смежных CEFR-уровней |
| Синонимы | Запрещены в A1-B2; в C1-C2 допустимы (нюансы языка) |
| Однокоренные | Запрещены в A1-B2; в C1-C2 допустимы |

Богатая тема → три набора: «основы» (COMMON/UNCOMMON) · «продвинутый» (RARE) · «углублённо» (EPIC/LEGENDARY)

**При рефакторинге:** заменяй слово на другое подходящего уровня — не перевешивай метку `rarity`.
