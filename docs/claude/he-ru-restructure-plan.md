## Ограничения

- **Не трогать:** серия Immigrant 1–16, пресетные колоды
- **Один агент за раз** редактирует `WordRegistry.kt` и `WordDataVersion.kt`
- **Наборы одного языка не параллельно** — агенты не видят слова друг друга → дубли
- После каждой темы: `python scripts/validate/find_real_dupes.py` → 0 ошибок перед коммитом
- Наборы типа «Иврит: …» — переименовать в тематический формат при переработке
